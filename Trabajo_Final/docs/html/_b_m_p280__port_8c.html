Descripción detallada

Implementación de la interfaz SPI de bajo nivel para el sensor BMP280.

Autor
Ing. Lucas Kirschner
Fecha
2 Apr 2025

Este archivo implementa las funciones de lectura, escritura y transferencia SPI que permiten la comunicación entre un microcontrolador STM32 y el sensor BMP280.

◆ BMP280_Port_Init()

bmp280_port_status_t BMP280_Port_Init ( void )

Inicializa la interfaz usada por el BMP280.

Esta función puede incluir configuraciones específicas del periférico SPI si fuera necesario. En esta implementación, se asume que la inicialización se realiza externamente mediante la HAL.
