Implementación de funciones para controlar el LCD HD44780.

Autor
Lucas Kirschner
Fecha

Este archivo contiene las definiciones de las funciones necesarias para controlar el display LCD basado en el controlador HD44780, incluyendo la inicialización, escritura de caracteres y limpieza del display.

◆ HD44780_Init()

hd44780_status_t HD44780_Init ( void )

Inicializa el controlador LCD HD44780 en modo de 4 bits.

Inicializa el controlador LCD HD44780.

Esta función realiza la secuencia de inicialización recomendada por la hoja de datos del HD44780 para configurar la interfaz en modo de 4 bits. Aunque se usan sólo los pines D4-D7, el controlador LCD arranca por defecto en modo de 8 bits. Por lo tanto, es necesario enviar tres veces el comando Function Set (0x03) como si se tratara de una interfaz de 8 bits.

Dado que sólo están conectados los pines D4-D7, se transmite únicamente el nibble alto 0x03, el cual el LCD interpreta como 0x30 (modo 8 bits). El LCD asume que el nibble bajo es 0 porque no hay señal conectada en D3-D0.

Luego, se envía el nibble 0x02 para indicar que se usará modo 4 bits de forma definitiva. A partir de ese punto, todos los comandos se envían en dos nibbles (alto y bajo).

La secuencia incluye los siguientes pasos:

  1. Inicialización del periférico
  2. Espera inicial tras el encendido (más de 15 ms después de que VCC haya subido a 4.5V).
  3. Envío de tres comandos para configurar el controlador en modo de 8 bits.
  4. Cambio a modo de 4 bits.
  5. Configuración de parámetros de operación como número de líneas y tipo de fuente.
  6. Apagado del display.
  7. Limpieza del display.
  8. Encendido del display.
  9. Configuración del modo de entrada (incremento de la dirección y sin desplazamiento de la pantalla).
Nota
Es importante respetar los tiempos mínimos entre comandos para asegurar la correcta inicialización del LCD.
