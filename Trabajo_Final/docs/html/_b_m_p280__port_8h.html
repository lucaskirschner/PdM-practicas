Descripción detallada

Interfaz de bajo nivel para la comunicación SPI con el sensor BMP280.

Autor
Ing. Lucas Kirschner
Fecha
2 Apr 2025

Este archivo define las funciones de lectura, escritura y transferencia SPI utilizadas para comunicar el sensor BMP280 con un microcontrolador STM32.

◆ BMP280_Port_Init()

bmp280_port_status_t BMP280_Port_Init ( void )

Inicializa la interfaz usada por el BMP280.

Esta función puede incluir configuraciones específicas del periférico SPI si fuera necesario. En esta implementación, se asume que la inicialización se realiza externamente mediante la HAL.
